# Papermc-downloader

Papermc-downloader is a JS CLI for download papermc 1.17.1 and all versions of Papermc, also download multiples paths.

## Installation

Use the npm install papermc-downloader to install this.

```bash
npm -g install papermc-downloader
```

## Usage

Execute the command and enter the requested data

```bash
papermc-downloader
```
